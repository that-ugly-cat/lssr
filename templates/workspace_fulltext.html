Fetch

For every screening-1 included record with a DOI, LSSR walks a ladder of open-access sources and stops at the first that yields real full text:

  1. Europe PMC — if the article is in PMC, its clean full-text XML is taken directly. No PDF, no conversion needed: these go straight to converted.
  2. Unpaywall & OpenAlex — every open-access location, repository copies before publisher ones, direct PDFs before landing pages.
  3. Landing pages — read for the PDF link publishers embed (citation_pdf_url).
  4. Publisher APIs — Elsevier / Springer / Wiley, only for their own DOIs and only if you've set the key in your profile.

A downloaded PDF is stored for step 2. What no route reaches keeps its best open-access link, so you can grab it by hand and upload it below.

Convert

Every stored PDF is sent to the paper2md service, which returns clean markdown with references, figures and tables stripped — the shape the assessment step reads. Records fetched as XML from Europe PMC are already converted and skip this step.

If paper2md is unreachable or rejects a file, the PDF is kept and the record stays fetchable for a retry — the run reports why. Manual uploads of Markdown, TXT or DOCX are already text, so they don't pass through here.
